# ─────────────────────────────────────────────────────────────────────────────
# HEAVY subtitle-OCR stack for Step 1 of the full video pipeline.
# Installed ONLY in the Docker image (streamlit/Dockerfile), NOT on Streamlit
# Community Cloud.
#

# installed in the image for it).
#
# Local full-pipeline dev:  pip install -r requirements.txt -r requirements-ocr.txt
# GPU host: replace `paddlepaddle` with `paddlepaddle-gpu` and enable GPU in the UI.
# ─────────────────────────────────────────────────────────────────────────────
